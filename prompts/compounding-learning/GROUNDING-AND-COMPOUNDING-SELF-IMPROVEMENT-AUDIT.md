# Grounding and Compounding Self-Improvement Audit

Audit the work against the complete compounding loop:

```text
source evidence
→ explicit object or decision
→ correct owner and destination
→ bounded change
→ verification and receipt
→ later reuse
→ measured outcome
→ retained correction
```

Evaluate:

- grounding quality and source freshness;
- whether sandbox/chat value escaped ephemeral custody;
- reuse of existing systems before new construction;
- duplicate and source-of-truth control;
- handoff recovery quality;
- correction persistence;
- tool selection and provider verification;
- operator burden and token cost;
- whether outputs changed future behavior;
- whether success and failure generated reusable prompts, tests, policies or scripts.

Return strengths, systemic failure modes, concrete amendments, metrics, and one smallest vertical slice that can prove improvement.
